## [k8s] - kubernetes object & api
### required field
* apiVersion
    - which version of the k8s api used to create this object
* kind
    - what kind of object to create
* metadata
    - data helps uniquely identify the object
        - name
        - namespace
        - UID (?)
* spec
    - state desired for the object
