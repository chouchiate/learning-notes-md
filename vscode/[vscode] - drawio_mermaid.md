## [vscode] - drawio / mermaid

### guide

```bash
git clone --recursive https://github.com/nopeslide/drawio_mermaid_plugin.git
cd drawio_mermaid_plugin/drawio_desktop
npm install
npm run build
cd ../vscode
npm install
npm run vscode:package

```
