## Docker_General_Note

### docker logs (live)

```bash
$ docker logs <container_id> -f
```

### List Images

```bash
$ docker image ls					
```

### List Containers

```bash
$ docker ps	
```

### Stop, Kill, Remove Container

```bash
$ docker stop <container Id>
$ docker rm <container Id>
```

### Entering Container Bash

```bash
$ docker exec -it <container_name> bash
```
